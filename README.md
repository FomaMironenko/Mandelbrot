# Mandelbrot fractal
This is a GUI project for Mandelbrot set investigation and visualization. 


## Control keys
***w*** - moving upwards<br/>
***s*** - moving downwards<br/>
***d*** - moving right<br/>
***a*** - moving lrft<br/>
